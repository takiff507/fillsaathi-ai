Privacy-first product rule

AI guides the user. It does not need private numbers.

FillSaathi AI should never ask users to send real bank details, identity numbers, passwords, OTPs, card details, UPI PINs, or full documents with visible sensitive data.

  • Keep API keys only on backend environment variables.
  • Mask sensitive details before sending anything to AI.
  • Do not store uploaded forms unless the user clearly agrees.
  • Give manual-fill warnings for high-risk fields.
